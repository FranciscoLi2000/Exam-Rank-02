# Exam-Rank-02

以下是针对 **Exam Rank 02** 各层级题目的中文解析与实现思路，帮助你高效应对考试：

---

### **Level 1: 基础字符串与逻辑处理**
#### 1. `first_word`
- **目标**：提取字符串的第一个单词（以空格/制表符分隔）。
- **关键点**：
- 跳过前导空格/制表符。
- 输出第一个连续非空字符序列。
- 参数校验：若参数数量≠1 或字符串无单词，输出换行符。
- **示例**：
```c
	// 输入: "   Hello World" → 输出: "Hello"
```

#### 2. `fizzbuzz`
- **目标**：输出1到100，3的倍数替换为`fizz`，5的倍数替换为`buzz`，同时满足则替换为`fizzbuzz`。
- **关键点**：
- 循环1到100，按条件判断输出。
- 使用`write`逐行输出，避免缓冲区问题。
- **示例**：
```c
	// 输入: 无 → 输出: "1\n2\nfizz\n4\nbuzz\n...\n"
```

#### 3. `repeat_alpha`
- **目标**：将每个字母重复其字母表位置的次数（例如`a→1次`，`b→2次`）。
- **关键点**：
- 区分大小写（`'a'`和`'A'`视为不同）。
- 非字母字符直接输出。
- 参数校验：若参数数量≠1，输出换行符。
- **示例**：
```c
	// 输入: "Ab" → 输出: "Abbb"
```

#### 4. `rev_print`
- **目标**：逆序输出字符串。
- **关键点**：
- 计算字符串长度，反向遍历输出。
- 参数校验：若参数数量≠1，输出换行符。
- **示例**：
```c
	// 输入: "abc" → 输出: "cba"
```

#### 5. `rot_13` 与 `rotone`
- **目标**：
- `rot_13`：每个字母替换为13位后的字母（`a→n`, `z→m`）。
- `rotone`：每个字母替换为下一个字母（`a→b`, `z→a`）。
- **关键点**：
- 处理字母循环（如`'z'+1→'a'`）。
- 非字母字符保持不变。
- **示例**：
```c
	// rot_13输入: "abc" → 输出: "nop"
	// rotone输入: "xyz" → 输出: "yza"
```

#### 6. `search_and_replace`
- **目标**：将字符串中的指定字符替换为另一字符。
- **关键点**：
- 参数校验：若参数数量≠3，输出换行符。
- 若替换字符未找到，直接原样输出。
- **示例**：
```c
	// 输入: "apple" 'p' 'o' → 输出: "aople"
```

#### 7. `ulstr`
- **目标**：反转字符串中字母的大小写。
- **关键点**：
- 使用位运算转换大小写（例如`c ^ 32`）。
- 非字母字符保持不变。
- **示例**：
```c
	// 输入: "Hello!" → 输出: "hELLO!"
```

### **Level 2: 进阶字符串与算法**
#### 1. `alpha_mirror`
- **目标**：将字母替换为其镜像字母（`a↔z`, `b↔y`）。
- **关键点**：
- 计算镜像位置：`c = 'a' + 'z' - c`（小写同理）。
- **示例**：
```c
	// 输入: "abc" → 输出: "zyx"
```

#### 2. `camel_to_snake` 与 `snake_to_camel`
- **目标**：
- `camel_to_snake`：将驼峰命名转换为下划线命名（`helloWorld→hello_world`）。
- `snake_to_camel`：将下划线命名转换为驼峰命名（`hello_world→helloWorld`）。
- **关键点**：
- 遍历字符串，检测大写字母或下划线，插入分隔符或转换大小写。
- **示例**：
```c
	// camel_to_snake输入: "hereIsACamel" → 输出: "here_is_a_camel"
```

#### 3. `ft_strcspn` 与 `ft_strspn`
- **目标**：
- `ft_strcspn(s, reject)`：返回`s`中**不包含**`reject`任何字符的初始段长度。
- `ft_strspn(s, accept)`：返回`s`中**只包含**`accept`字符的初始段长度。
- **关键点**：
- 双循环遍历字符，匹配终止条件。
- **示例**：
```c
	// ft_strcspn("abcde", "cd") → 返回值2（"ab"的长度）
```

#### 4. `inter` 与 `union`
- **目标**：
- `inter`：输出两字符串的交集字符（按第一字符串顺序，去重）。
- `union`：输出两字符串的并集字符（按命令行参数顺序，去重）。
- **关键点**：
- 使用哈希表或数组记录字符出现情况。
- 遍历时检查重复。
- **示例**：
```c
	// inter输入: "padinton" "paqefwtdjetyiytjneytjoeyjnejeyj" → 输出: "padinto"
```

#### 5. `is_power_of_2`
- **目标**：判断一个数是否为2的幂。
- **关键点**：
- 利用位运算：`n > 0 && (n & (n - 1)) == 0`。
- **示例**：
```c
	// 输入: 8 → 返回1（是2的幂）
```

#### 6. `print_bits` 与 `reverse_bits`
- **目标**：
- `print_bits`：逐位输出一个字节的二进制表示。
- `reverse_bits`：反转一个字节的位序（例如`0b10000000→0b00000001`）。
- **关键点**：
- 使用位掩码（`0x80`）逐位提取。
- 反转时通过位移和或运算重组。
- **示例**：
```c
	// print_bits(2) → 输出"00000010"
	// reverse_bits(0b00101100) → 返回0b00110100
```

### **Level 3: 数学与复杂逻辑**
#### 1. `add_prime_sum`
- **目标**：计算≤给定数的所有素数之和。
- **关键点**：
- 实现高效素数判断（试除法到平方根）。
- 处理边界：参数非正整数时输出0。
- **示例**：
```c
	// 输入: 5 → 输出: 10（2+3+5）
```

#### 2. `epur_str` 与 `expand_str`
- **目标**：
- `epur_str`：清理字符串中的多余空格，保留单词间单个空格。
- `expand_str`：单词间用三个空格分隔。
- **关键点**：
- 遍历字符串，跳过连续空格，记录单词边界。
- **示例**：
```c
	// epur_str输入: "   Hello   World  " → 输出: "Hello World"
```

#### 3. `ft_range` 与 `ft_rrange`
- **目标**：生成连续整数数组。
- **关键点**：
- 计算数组长度：`end - start + 1`（注意正负）。
- 动态分配内存（`malloc`），填充值。
- **示例**：
```c
	// ft_range(1, 3) → 返回数组[1,2,3]
	// ft_rrange(1, 3) → 返回数组[3,2,1]
```

#### 4. `hidenp`
- **目标**：判断字符串`s1`是否是`s2`的子序列（字符顺序一致，可不连续）。
- **关键点**：
- 双指针遍历，匹配`s1`的每个字符。
- **示例**：
```c
	// 输入: "abc" "a1b2c3" → 输出1（是子序列）
```

### **Level 4: 高级算法（泛洪填充）**
#### `flood_fill`
- **目标**：从指定点开始，用`'F'`填充连通区域。
- **关键点**：
- 使用递归或队列实现泛洪算法。
- 检查边界条件（数组越界、字符是否匹配）。
- **示例**：
```c
	// 输入: 二维数组和起始点(2,2) → 填充所有相邻相同字符为'F'
```

### **通关策略**
1. **分模块实现**：将功能拆分为独立函数（如素数判断、字符串遍历）。
2. **边界测试**：覆盖空输入、极端值、非法参数等情况。
3. **内存管理**：动态分配后检查指针，确保无泄漏（`valgrind`检测）。
4. **效率优化**：避免重复计算（如素数判断缓存结果）。
